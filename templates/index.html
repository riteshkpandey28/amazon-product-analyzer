How the System Work ?

  1. The System takes the Product ID as an Input.
  2. The System performs web scrapping and gathers review information about the product whose ID is given as input.
  3. Around 500-800 Reviews are scrapped at a time.
  4. Then the System Performs Sentimental Analysis on the scrapped review using Bert Sentimental Machine Learning Model.
  5. All Reviews are classified into Positive and Negative.
  6. Later, the positive and negative reviews are passed through LDA Model and accordingly feature analysis is performed.
  7. Separate Graphs of liked and disliked feature are obtained from Positive and Negative Reviews respectively.
